# deepgram-vapi-call-sample

## Setting up your Vapi account
- [Sign Up](https://vapi.ai)
- Create an Account to get an API Key and create a new Voice Bot

## Set Env Var

```
export VAPI_API_KEY=
export VAPI_PHONE_NUMBER_ID=
export VAPI_ASSISTINT_ID=
```

## Setup

```
npm i
```

## Running

```
npm run start
```

## Using

```
http://localhost:3000
```
