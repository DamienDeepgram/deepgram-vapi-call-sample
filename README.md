# deepgram-vapi-call-sample

## Accounts Required

- [Deepgram](https://deepgram.com)
- [Vapi](https://vapi.ai)


## Set Env Var

```
export VAPI_API_KEY=
export VAPI_PHONE_NUMBER_ID=
export VAPI_ASSISTINT_ID=
```

## Setup

```
npm i
```

## Running

```
npm run start
```

## Using

```
http://localhost:3000
```
